# Cómo instalar — La forma fácil (sin saber programar)

**No tenés que escribir comandos. Tu IA lo hace todo.**

Hay 3 formas de instalar el Mothership Method. Elegí la que más te guste.

---

## Forma 1 — Pedile a tu IA que lo haga

Esta es **la más fácil**. Funciona con Claude Code, Codex, Gemini CLI, OpenClaw, o cualquier agente IA que use terminal.

### Pasos

1. Abrí tu agente IA favorito (Claude Code, Codex, Gemini CLI, OpenClaw, etc.)

2. Copiá ESTE TEXTO entero (sin cambios) y pegalo en el chat:

```
Por favor instalá el "Mothership Method" en mi máquina.

Pasos:
1. Hacé git clone https://github.com/mattzz123/mothership-starter.git en mi home
   (si no podés git clone, descargá el ZIP/tar.gz desde la página del repo)
2. Entrá al directorio mothership-starter
3. Leé el archivo BOOTSTRAP_FOR_AGENT.md ENTERO
4. Seguí los pasos exactos que describe ese archivo
5. Cuando termines, mostrame en 3 puntos cómo voy a usar el sistema

Soy un usuario sin conocimiento técnico. Si algo falla, explicame en mis palabras qué pasó. Si necesitás permiso para algo, pedímelo antes.
```

3. Presioná Enter. La IA hace el resto.

4. Cuando termine, te va a explicar cómo usar el sistema. Eso es todo.

---

## Forma 2 — Un comando en tu terminal

Si te animás a abrir terminal:

```bash
curl -fsSL https://raw.githubusercontent.com/mattzz123/mothership-starter/main/install.sh | bash
```

El instalador te va a hacer 1-2 preguntas. Acepta los defaults presionando Enter.

---

## Forma 3 — Descarga manual

1. Descargá el ZIP: `https://github.com/mattzz123/mothership-starter/archive/main.zip`
2. Descomprimí en cualquier carpeta.
3. Abrí terminal en esa carpeta.
4. Ejecutá: `bash install.sh`

---

## ¿Qué pasa después de instalar?

Tu IA queda con **memoria persistente entre sesiones**. La próxima vez que la abras, va a recordar lo que hablaron — sin que vos le tengas que volver a explicar nada.

Lectura recomendada (5 minutos):
- `CHEATSHEET.md` — los 10 comandos esenciales (1 página)

Soporte:
- WhatsApp/email del onboarding (incluido en tu compra)
- Discord del grupo de usuarios

---

## ¿Qué hacer si algo sale mal?

1. Pedile a tu IA: *"Algo falló durante la instalación de Mothership Method. Mostrame el error y arreglemoslo paso a paso."*
2. Si la IA no puede arreglarlo sola, sacá una captura del error y mandala por WhatsApp al soporte.
3. **No borres nada manualmente.** El sistema hace backups automáticos.
