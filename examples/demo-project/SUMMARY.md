# mi-primer-proyecto — SUMMARY

## Scope
Proyecto demo. Sirve para que experimentes cómo se comporta la IA cuando tiene un bundle correctamente estructurado.

## Arquitectura
- Bundle estándar del Mothership Method (5 archivos + META.json).
- Sin runtime propio.
- Sin servicios externos.
- Nada sensible — es ejemplo puro.

## Decisiones estables
- Formato: markdown plano.
- Single-agent suficiente (no requiere coordinación multi-agente para el demo).

## Guardrails
- No hacer cambios sin que el usuario los pida.
- Seguir el Doc Checklist de 9 pasos después de cambios.
